Styleguide

This is a basic Styleguide, based on modern web-techonlogies (like HTML5 and CSS3) build with the rock solid frontend framework Bootstrap 3 SCSS.
Static Photoshop webdesigns maybe looking nice, but we want to design websites for the next generation!
Please use this as base for new Projects.

A website is not a nice picture or catalog-like-layout - it's a content based product you can use and explore!
This content based product should be usable on all devices, which can access a website.
We design in the Browser and use software like Photoshop only for the necessary image optimazion and graphics,
because we want to design & develop great web-based experiences.

Our stragetegy


MOBILE FIRST WEBDESIGN

"Designing for mobile first can not only open up new opportunities for growth, it can lead to a better overall user experience for a Website"
LUKE WROBLEWSKI - Mobile first

"Thinking mobile first forces you to only keep what matters in websites. We have to think about usability & content because people on mobile have very little time and screen space.
BRAD FROST - a future friendly web

"Get your content ready to go anywhere because it’s going to go everywhere."
SARA WACHTER-BOETTCHE - Future-Ready Content

"You don’t get to decide which platform or device your customers use to access your content: they do."
PAUL FORD - Content strategy for mobile
